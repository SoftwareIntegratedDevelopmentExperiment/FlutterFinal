# 移动应用软件开发实训期末作业

**ourapplication是我们的项目程序**，剩下的是谷歌的机器学习模型的api。使用方法即在ourapplication中点击lib，点击main.dart，点击run即可进行代码的运行

**注意：当前代码仅支持安卓系统的运行，苹果以及windows均无法使用该程序，请用安卓机进行app的下载**
